# mars-photo-search-app

This is an app to help people search for pictures from the Curiousity rover on Mars. You can choose the Sol date and which camera.

Dependencies:
 - flask
 - flask-cors
 - requests
 - gunicorn
 - react
 - gatsby
 - axios
 - formik
 - yup
 - query-string
 - bootstrap
 - reactstrap
 - prettier
 - husky
 - pretty-quick
